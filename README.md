# dabinj.github.io

Chirpy 기반 Jekyll 블로그입니다.

## 로컬 실행

```bash
BUNDLE_USER_HOME=.bundle-home BUNDLE_PATH=vendor/bundle BUNDLE_FORCE_RUBY_PLATFORM=true bundle install
BUNDLE_USER_HOME=.bundle-home BUNDLE_PATH=vendor/bundle BUNDLE_FORCE_RUBY_PLATFORM=true bundle exec jekyll serve
```

## 참고

- 테마 기반: `jekyll-theme-chirpy`
