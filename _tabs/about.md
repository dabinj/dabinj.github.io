이 블로그는 생물정보학 분석 과정에서 남기는 기록을 정리하기 위해 만들었습니다.

실험 메모, 파이프라인 구축 과정, 문제 해결 과정, 그리고 프로젝트별 회고를 꾸준히 쌓아갈 예정입니다.

## 다루는 내용

- NGS 분석 및 파이프라인 정리
- Jupyter, Python, Bash 기반 실험 기록
- GitHub 기반 프로젝트 운영 메모
- 논문과 기술 문서 요약

## 운영 방식

콘텐츠는 Jekyll 기반 Markdown 포스트로 관리합니다.
글을 파일로 다루기 때문에 버전 관리와 배포 흐름이 단순합니다.
